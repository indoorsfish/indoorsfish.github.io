HTML知识点整理

2018年1月13日

基本知识点：

概念

什么是HTML?

  • HTML 超文本标记语言:（HyperText Markup Language）是用来描述网页的一种语言
  • HTML 不是一种编程语言，而是一种标记语言，拥有一套标记标签
  • HTML 文档包含了HTML 标签及文本内容

简单地说，现在网络上绝大部分网页都是基于HTML编写的。

HTML编辑器

  • Sublime Text 3 (新手级)
  • WebStorm (高手级)
  • Text文本文档 (骨灰级)
  • 等等

HTML基本结构

<!DOCTYPE html>   -------------- 文档定义类型
    <head>  -------------- 文档头部标记：含脚本，样式表等等。
        <meta />  -------------- 定义文档字符集、使用语言、作者等基本信息
        <title>标题</title>  -------------- 定义文档的标题
    </head>
    <body>   -------------- 网页主体,是可见的页面内容
        <h1>Hello world</h1> -----------标题
   </body>
</html>

HTML元素

HTML 元素指的是从开始标签（start tag）到结束标签（end tag）的所有代码。分为：

1. 标题 段落 块

//标题:h1~h6
<h1>1号标题</h1>
<h2>2号标题</h2>
<h3>3号标题</h3>
<h4>4号标题</h4>
<h5>5号标题</h5>
<h6>6号标题</h6>

//段落：<p></p>
<p>第一段</p>
<p>第二段</p>

//块：<div></div>
<div>
<h1>标题</h1>
<p>段落</p>
</div>

*有哪些块元素？哪些内嵌元素？

块元素：块级元素在浏览器显示时，通常会以新行来开始（和结束） display:block

例子：<h1>, <p>, <ul>, <table>，<div>

内嵌元素：内联元素在显示时通常不会以新行开始。 display:inline

例子：<b>, <td>, <a>, <img>,<span>


2. 图片 超链接 锚点链接

绝对路径和相对路径的区别(网络地址为例)
 绝对路径：协议+主机+路径+文档
 相对路径：/根目录  ../上一级目录  ../../上上一级目录

//图像:<img src=""  alt=""/>

//超链接：<a href=""></a>
<a href="http://www.baidu.com" target="_blank"></a>

//锚点链接:定义一个唯一的'id' 赋给href 实现跳转
<a href="#p1">点击前往第六段</a>
<p id="p1">第六段</p>

*url、src与href属性的区别:

href 用于建立链接关系，<a<, <link<元素，浏览器在解析到此处时之道这里是个外部链接（或锚点）而并不需要加载这个链接的内容。因此页面解析不会被暂停。不载入文档

src用于置换元素，即是说这个元素的内容通过src的地址获得，不受css样式限制。<script /> <img /><video /> 等.浏览器在解析到此处时会暂停所有渲染，加载直到此处内容被加载完成。载入文档

url狭义来讲，在这里url代表了CSS的一个数据类型，它指定了一个数据的链接位置。通常的用法是用url()函数:background-image:url(images/01.jpg);

url不是属性，src和href是属性

4. 列表: 有序列表 无序列表 定义列表

有序列表：通常用来表示内容之间的顺序或者是重要性关系，每一个列表都分为多个子项，每一个子项都有相应的编号。
<ol>
  <li>新闻1</li>
  <li>新闻2</li>
  <li>新闻3</li>
  <li>新闻4</li>
</ol>

无序列表：仅仅用于表示内容之间存在有并列关系，也是分为多个子项，但是子项之前没有相应的编号，只有一个原点用于标识。
<ul>
  <li>...</li>
  <li>...</li>
  <li>...</li>
</ul>

定义列表:通常用于表示名词或者是概念的定义，每一个子项有两个部分组成，第一部分是名词或者是概念，第二部分是相应的解释和描述。
<dl>
  <dt>...</dt>
  <dd>...</dd>
</dl>

5.表格: table

//table元素的作用是显示表格化的数据！
//主要元素：table tr th td   （隐藏元素：tbody thead tfoot）

//语法书写：
<table>  ---------表格
  <tr>   ---------行
    <th>内容</th>  ---------单元格(字体加粗)
    <td>内容</td>  ---------单元格
    <td>内容</td>
  </tr>
</table>

//其他：
合并单元格：rowspan="2"合并行，colspan="3"和并列 
表格边框设置：1.table中border-spacing 2.border-collapse=collapse


6.表单: form

//表单对于用户而言是数据的录入和提交的界面；
//表单对于网站而言获取用户信息的途径。

基本格式：<form action="路径">
			...
	  	  </form>

常用的表单元素:

<input type=""/>  //根据不同的 type 属性，可以变化为多种形态。
<textarea></textarea>  //定义多行输入字段（文本域）。
<button></button> //定义可点击的按钮。
<select>  //定义下拉列表。
	<option>1</option>
	<option>2</option>
</select>   

文本框：<input type="text"> 定义供文本输入的单行输入字段。
<input type="text" name="控件名字" value="值" maxlength="最大输入字符长度" size="控件宽度" readonly="readonly"（只读） />

密码框：<input type="password"> 定义密码字段。
<input type="password" name="控件名字" value="值" maxlength="最大输入字符长度" size="控件宽度" readonly="readonly"（只读） />

多选勾选控件：<input type="checkbox"> 定义复选框
<input type="checkbox" name="控件名字" value="值" checked="checked"(已选中) disabled = "disabled"(禁用控件) />

单选勾选控件：<input type="radio"> 定义单选按钮。
<input type="radio" name="控件名字" value="值" checked="checked"(已选中) disabled = "disabled"(禁用控件) />

提交表单按钮：<input type="submit"> 定义提交表单数据至表单处理程序的按钮。
<input type="submit" value="按钮字样" />

重置表单按钮：
<input type="reset" value="按钮字样" />

上传文件按钮：
<input type="file" name="文件名称">

按钮：<input type="button> 定义按钮。
<input type="button" onclick="alert('Hello World!')" value="Click Me!">

标签：<input type="radio" name="sex" id="male" > 
<label for="male"">男性</label>
