Bootstrap知识点整理

2018年2月20日

引：

Bootstrap，来自 Twitter，是目前最受欢迎的前端框架。Bootstrap 是基于 HTML、CSS、JAVASCRIPT 的，它简洁灵活，使得 Web 开发更加快捷。Bootstrap的设计目标是移动设备优先，然后才是桌面设备。

Bootstrap官网下载的文件主要有Bootstrap.css Bootstrap.js插件（依赖jquery） Bootstrap.font三个重要的文件，以下是基础模板：

<!DOCTYPE html>
<html>
   <head>
      <meta charset="UTF-8">
      <title>Bootstrap 模板</title>
      <meta http-equiv="X-UA-Compatible" content="IE=edge">  //在ie中运行最新的渲染模式
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <!-- 引入 Bootstrap -->
      <link href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css" 
rel="stylesheet">
      <!-- HTML5 Shiv 和 Respond.js 用于让 IE8 支持 HTML5元素和媒体查询 -->
      <!-- 注意： 如果通过 file://  引入 Respond.js 文件，则该文件无法起效果 -->
      <!--[if lt IE 9]>
         <script src="https://oss.maxcdn.com/libs/html5shiv/3.7.0/html5shiv.js"></script>
         <script src="https://oss.maxcdn.com/libs/respond.js/1.3.0/respond.min.js"></script>
      <![endif]-->
   </head>
   <body>
      <h1>Hello, world!</h1>

      <!-- jQuery (Bootstrap 的 JavaScript 插件需要引入 jQuery) -->
      <script src="https://code.jquery.com/jquery.js"></script>
      <!-- 包括所有已编译的插件 -->
      <script src="js/bootstrap.min.js"></script>
   </body>
</html>

Bootstrap样式篇

容器div

  • .container //居中固定的布局
  • .container-fluid //响应的自适应大小布局

文本

基本样式：

  • 行高为1.42857143（line-height），大约是20px
  • 颜色为深灰色（#333）
  • 全局文本字号为14px(font-size)
  • 字体为"Helvetica Neue", Helvetica, Arial, sans-serif;（font-family）

颜色：

  • .text-muted：提示，使用浅灰色（#999）
  • .text-primary：主要，使用蓝色（#428bca）
  • .text-success：成功，使用浅绿色(#3c763d)
  • .text-info：通知信息，使用浅蓝色（#31708f）
  • .text-warning：警告，使用黄色（#8a6d3b）
  • .text-danger：危险，使用褐色（#a94442）

对齐：

  • .text-left：左对齐
  • .text-center：居中对齐
  • .text-right：右对齐
  • .text-justify：两端对齐

其他：

  • .lead: 段落添加强调文本 ,让其显示效果显著
  • 重新定义了small、strong、em 、abbr(缩写)等标签

列表

  • .list-unstyled :去除默认的列表样式的风格
  • .list-inline :实现内联列表，把所有的列表项放在同一行中。为制作水平导航而生。

表格

在table标签中添加一下类：

  • .table：基础表格 //其他类必须和此类配合使用
  • .table-striped：斑马线表格
  • .table-bordered：带边框的表格
  • .table-hover：鼠标悬停高亮的表格
  • .table-condensed：紧凑型表格

在tr标签中添加一下类：

  • .active :表示当前活动的信息
  • .success :表示成功或者正确的行为
  • .info :表示中立的信息或行为
  • .danger :表示危险或者可能是错误的行为
  • .table-responsive：响应式表格

响应式表格：

table作为子容器，div作为父容器 div class="table-responsive"

表单

创建基本表单 <form> 的规则：

  1. 向父 <form> 元素添加 role="form"。
  2. 把标签和控件放在一个带有 class .form-group 的 <div>//.form-group 相当于网格系统中的“row” 每行放一个form.group的div 中。这是获取最佳间距所必需的。
  3. 向所有的文本元素 <input text>、<textarea> 和 <select>添加 class ="form-control" //.form-control让控件在各种表单风格中样式不出错

分类：

  • 水平表单： .form-horizontal 实现水平表单效果
  • 内联表单： .form-inline 表单的控件都在一行内显示

控件尺寸：

  • .input-sm:让控件比正常大小更小 //input标签中,高度变化
  • .input-lg:让控件比正常大小更大

验证状态：

    和font-group放在行包裹form组件div
  • .has-warning:警告状态（黄色）
  • .has-error：错误状态（红色）
  • .has-success：成功状态（绿色）
  • .has-feedback 后面加图标，和123放一起

图片

  • .img-responsive：响应式图片
  • .img-rounded：圆角图片
  • .img-circle：圆形图片
  • .img-thumbnail：缩略图片

辅助类

  • .pull-left：元素浮动到左边
  • .pull-right：元素浮动到右边
  • .center-block：设置元素为 display:block 并居中显示
  • .clearfix：清除浮动 配合使用
  • <span class="caret"></span>显示下拉式功能,三角形
  • <label class="sr-only" for="pass">密码</label> 除了屏幕阅读器外，其他设备上隐藏元素,用于屏幕阅读器的，帮助残障人士更好的访问网站

网格系统

Bootstrap 提供了一套响应式、移动设备优先的流式网格系统，随着屏幕或视口（viewport）尺寸的增加，系统会自动分为最多12列（默认）。

注：所有行必须放置在 .container class内，以便获得适当的对齐（alignment）和内边距（padding）。

网格选项：

  • .col-xs- ： 自动（超小屏）
  • .col-sm- ：750px（小屏）
  • .col-md- ：970px（中屏）
  • .col-lg- ：1170px（大屏
  • .col-lg- ：1170px（大屏

列偏移：

.col-md-offset-n 向右偏移n使得元素不紧挨着一起

用法：

<div class="container">	
	<h1>Hello, world!</h1>	
    <div class="row" >	
        <div class="col-xs-6 col-md-offset-3" 
        style="background-color: #dedef8;box-shadow: 
        inset 1px -1px 1px #444, inset -1px 1px 1px #444;">	
            <p>	示例文本
            </p>	
        </div>	
    </div>	
</div>

按钮

基本样式：

  • .btn 基础按钮 //其他按钮样式和此类配合使用
  • .btn-default 默认按钮
  • .btn-primary 主要按钮
  • .btn-success 成功按钮
  • .btn-info 信息按钮
  • .btn-warning 警告按钮
  • .btn-danger 危险按钮

按钮尺寸：

  • .btn-lg 大型按钮
  • .btn-sm 小型按钮
  • .btn-xs 超型按钮
  • .btn-block 块状按钮 width：100%

按钮状态：

  • .active 活动状态：悬浮状态(:hover)，点击状态(:active)和焦点状态（:focus）
  • .disabled 禁用状态（带有鼠标样式）

按钮组：

  • .btn-group 包裹所有按钮btn
  • .btn-toolbar 包裹所有按钮组btn-group
  • .btn-group-* 设置按钮组大小-lg -sm -xs
  • .btn-group-vertical 垂直分组

下拉菜单 (非select)

创建步骤：

  1. 使用一个名为“dropdown”的容器<div>包裹了整个下拉菜单元素
  2. 使用了一个 <button>按钮做为父菜单，并且定义类名“dropdown-toggle”和自定义“data-toggle”属性，且值必须和最外容器类名一致
  3. 下拉菜单项使用一个ul列表，并且定义一个类名为“dropdown-menu”下拉菜单默认是左对齐，如果你想让下拉菜单相对于父容器<ul>右对齐时，加一个“pull-right”或者 “dropdown-menu-right”类名
  4. <li>中添加类名.divider来实现添加下拉分隔线的功能; .dropdown-header 每个组添加一个头部（标题);菜单项状态:当前状态（.active）和禁用状（.disabled）
  5. class="caret" 三角形默认往下 若想往上向父 .btn-group 容器添加 .dropup 即可（按钮下拉）

示例：

<div class="dropdown">
    <button type="button" class="btn dropdown-toggle" id="dropdownMenu1" data-toggle="dropdown">主题
        <span class="caret"></span>
    </button>
    <ul class="dropdown-menu" role="menu" aria-labelledby="dropdownMenu1">
        <li role="presentation">
            <a role="menuitem" tabindex="-1" href="#">Java</a>
        </li>
        <li role="presentation">
            <a role="menuitem" tabindex="-1" href="#">数据挖掘</a>
        </li>
        <li role="presentation" class="divider"></li>
        <li role="presentation">
            <a role="menuitem" tabindex="-1" href="#">分离的链接</a>
        </li>
    </ul>
</div>

导航

基本样式（无序列表ul中的类）

  • .nav 基础
  • .nav-tabs 标签形导航
  • .nav-pills胶囊形（pills）导航
  • .nav-stacked 垂直堆叠的导航
  • .nav-justified 自适应导航指的是导航占据容器全部宽度

导航状态（li中的类）

  • .active默认选中项
  • .disabled禁用状态
  • data-toggle="tab"，则可以动态替换

导航栏

创建步骤：

  1. 向 <nav> 标签添加 class .navbar、.navbar-default， role="navigation"，有助于增加可访问性。
  2. 向 <div> 元素添加一个标题 class .navbar-header，内部包含了带有 class navbar-brand 的 <a> 元素。这会让文本看起来更大一号。
  3. 为了向导航栏添加链接，只需要简单地添加带有 class .nav、.navbar-nav 的无序列表ul即可。

响应式导航栏

  1. ul正常布局, 外面套了一层div,添加类名:collapse,navbar-collapse或navbar-response-collapse
  2. 保证在窄屏时要显示的图标样式（固定写法)：
    3. <button type="button" 
   class="navbar-toggle" data-toggle="collapse"  data-target="#example-navbar-collapse">
            <span class="sr-only">切换导航</span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
        </button>
<a class="navbar-brand" href="#"> logo </a>

导航栏定位

  • 固定定位： .navbar-fixed-top 导航条固定在浏览器窗口顶部； .navbar-fixed-bottom：导航条固定在浏览器窗口底部
  • 静态定位：.navbar-static-top 随着页面向下滚动而消失

反色导航条：.navbar-inverse

导航栏元素

  • 按钮 .btn .btn-default .navbar-btn
  • 文本 .navbar-text
  • 普通链接 .navbar-link
  • 表单 form .navbar-form .navbar-left 相左浮动

面包屑导航： ul.breadcrumb

分页

默认的分页：无序列表ul.pagination

分页样式：

  • 分页状态：li.disabled、li.active
  • 分页大小：ul.pagination-* 可以接-lg、-sm 、xs

翻页 略

标签和徽章

标签

基本标签： span.label

使用class label-default、label-primary、label-success、label-info、label-warning、label-danger 来改变标签的外观

徽章

基本徽章： span.badge

进度条

基本进度条：

  1. 添加一个带有 class .progress 的 <div>。
  2. 接着，在上面的 <div> 内，添加一个带有 class .progress-bar 的空的 <div>。
  3. 添加一个带有百分比表示的宽度的 style 属性，例如 style="width: 60%"; 表示进度条在 60% 的位置。

条纹进度条：

在 class .progress 的 <div>中添加 .progress-striped

动画进度条：

在 class .progress .progress-striped的 <div>同时添加.active。

媒体对象

基本媒体元素

  • 媒体对像的容器：常使用“media”类名表示，用来容纳媒体对象的所有内容
  • 媒体对像的对象：常使用“media-object”表示，就是媒体对象中的对象，常常是图片
  • 媒体对象的主体：常使用“media-body”表示，就是媒体对像中的主体内容，可以是任何元素，常常是图片侧边内容
  • 媒体对象的标题：常使用“media-heading”表示，就是用来描述对象的一个标题，此部分可选
  • 除了上面四个部分之外，在Bootstrap框架中还常常使用.media-left 或者.media-right 来控制媒体对象中的对象浮动方式

列表组

  • 向元素 <ul>添加 class .list-group。
  • 向 <li> 添加 class .list-group-item。

面板

基础面板：向 <div>元素添加 class .panel 和 class .panel-default

面板标题：

  • 使用 .panel-heading class 可以很简单地向面板添加标题容器。
  • 使用带有 .panel-title class 的 <h1>-<h6> 来添加预定义样式的标题。

面板脚注：把按钮或者副文本放在带有class .panel-footer 的 <div>中

带语境色彩的面板：使用语境状态类 panel-primary、panel-success、panel-info、panel-warning、panel-danger，来设置带语境色彩的面板。
