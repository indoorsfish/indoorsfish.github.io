CSS知识点整理

2018年1月13日

基本知识点：

CSS 指层叠样式表 (Cascading Style Sheets)

CSS构成：选择器，属性和属性值。

基本语法： selector {property: value;}

样式分类

  1. 内联样式：把CSS样式表写在HTML对应的标记内。 格式如下：<p style="font- size:14pt;color:blue">蓝色14号文字</p>
  2. 内部样式：把CSS样式表放到文档中： 格式如下：<style type=“text/css” >...</style>
  3. 外部样式：把编辑好的CSS文档保存成“.CSS”文件，然后在<head>中定义。 格式如下：<head> ...<link rel="stylesheet" type="text/css" href="mystyle.css">...</head>
  4. 导入样式：与链入外部样式的功能基本相同，只是语法和实现方式上有差别。 格式如下： <style> @import url(css.css);</style>

样式优先级：！important>内联样式>内部样式>外部样式
网页制作者》用户自定义》浏览器

选择器

  1. 类选择器 ：类选择器根据类名来选择，前面以”.”来命名，如.demo{color:#FF0000;}
  2. id选择器 ：根据元素ID来选择元素，具有唯一性。前面以”#”号来标志，如：#demo {color:#FF0000; }
  3. 标签选择器：HTML页面是有很多不同的标签组成，标签选择器，则是决定对应标签采用相应的CSS样式。
    如：p {color:#FF0000; }
  4. 伪类选择器(针对超链接): a:link未访问、a:visited已访问、a:hover鼠标悬停时、a:active链接被点击的时刻
  5. 后代选择器（派生选择器）:通过依据元素在其位置的上下文关系来定义样式，可以使标记更加简洁。
    如：#demo p {color:#ff0000; size:14px;} 表示ID为demo元素下的所有p元素，设置颜色#ff0000字体大小14px
  6. 通用选择器 通用选择器用*来表示。 如：*{font-size: 12px;}表示所有的元素的字体大小都是12px
  7. 并集选择器（群组选择器）当几个元素样式属性一样时，可以共同调用一个声明，元素之间用逗号分隔。
    如：p, td, li {line-height:20px; color:#ff0000;}

div四种position定位方法

  1. 静态定位 static:元素出现在正常的流中（忽略 top, bottom, left, right z-index 声明）
  2. 绝对定位 absolute:生成绝对定位的元素，相对于 static 定位以外的第一个祖先元素进行定位
  3. 固定定位 fixed:生成绝对定位的元素，相对于浏览器窗口进行定位（老IE不支持）
  4. 相对定位 relative:生成相对定位的元素，相对于其在普通流中的位置进行定位。

display 的值的作用

  1. block：像块类型元素一样显示。 可以设置width、height、margin和padding属性
  2. inline：像行内元素类型一样显示。使用width，height属性水平方向产生边距效果，但竖直方向不会产生边距效果。
  3. inline-block：像行内元素一样显示，但其内容象块类型元素一样显示。 可以设置padding margin border ，但不能设置height width
  4. list-item：像块类型元素一样显示，并添加样式列表标记。

CSS盒模型

css盒子模型 又称框模型 (Box Model) ，包含了元素内容（content）、内边距（padding）、边框（border）、外边距（margin）几个要素。主要分为以下两种：

一、标准w3c盒模型（默认）：设置模型的height和width即是设置content区的高度和宽度，与padding和border无关。

二、IE盒模型：模型的height和width即是content、padding和border高度和宽度之和。转化该模型可设置
*{box-sizing:border-box ;}。

页面布局方式

  1. 浮动布局：float:left/right;
  2. 凝胶布局：margin:auto;
  3. 定位布局：position:absolute;
  4. 表格

样式继承

可继承： font-size、font-family、color、text-indent等

不可继承 ：border、padding、margin、width、height 等


注意事项：

  1. 如果属性值是若干单词，则要给值加上引号比如：p{font-family:Georgia,"sans serif",Palatino;}
  2. CSS语法书写对大小写不敏感但是元素在调用id和class时区分大小写的。
  3. vertical-align图片与文字（内容）垂直对齐 text-align文字（内容）水平对齐
  4. 在一个div里，如何让内容垂直居中？ 方式是设置line-height属性，属性值为div的高度。
  5. 在ul无序列表里，让内容居左对齐。 设置ul的margin:0px;padding:0px
  6. Div如何居中？ 设置margin:0 auto;
